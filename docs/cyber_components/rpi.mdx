# Raspberry Pi, a Single Board Computer (SBC), for ThingsBoard Deployment

Deploying ThingsBoard on the Raspberry Pi 4 Model B leverages this single-board computer (SBC) as the primary hardware platform, serving as a local server within edge, fog, and cloud computing layers. This configuration utilizes the Raspberry Pi 4's capabilities to manage key IoT operations such as decentralized data processing, OTA updates, and mobile app integration, critical for enhanced data handling and security.

### Advantages of Raspberry Pi Deployment

Deploying ThingsBoard on Raspberry Pi 4 offers significant benefits:

- **Complete Data Control**: Provides full governance over data, ensuring secure management of sensitive or proprietary information.
- **Reduced Operational Costs**: Minimizes reliance on cloud services, significantly reducing operational expenses.
- **Enhanced Processing Speed and Reliability**: Local data processing on Raspberry Pi improves response times and system stability, facilitating immediate actions based on real-time data insights.

### Technical Specifications of Raspberry Pi 4 Model B

The Raspberry Pi 4 Model B is equipped with enhanced features that boost its performance, suitable for demanding tasks and applications:

- **Processor**: Broadcom BCM2711, Quad-core Cortex-A72 (ARM v8) 64-bit SoC @ 1.8GHz
- **Memory Options**: 1GB, 2GB, 4GB, or 8GB LPDDR4-3200 SDRAM
- **Wireless Connectivity**:
  - Dual-band 2.4 GHz and 5.0 GHz IEEE 802.11ac wireless
  - Bluetooth 5.0, BLE
- **Networking**: Gigabit Ethernet
- **USB Ports**: 2 USB 3.0 ports; 2 USB 2.0 ports
- **GPIO**: Standard 40-pin GPIO header, fully backwards compatible with previous boards
- **Video Output**:
  - 2 × micro-HDMI® ports (up to 4kp60 supported)
- **Camera Interface**: 2-lane MIPI CSI camera port
- **Audio**: 4-pole stereo audio and composite video port
- **Video and Graphics**:
  - H.265 (4kp60 decode), H264 (1080p60 decode, 1080p30 encode)
  - OpenGL ES 3.1, Vulkan 1.0
- **Storage**: Micro-SD card slot for loading operating systems and data storage
- **Power Options**:
  - 5V DC via USB-C connector (minimum 3A)
  - 5V DC via GPIO header (minimum 3A)
  - Power over Ethernet (PoE) enabled (requires separate PoE HAT)
- **Operating Temperature**: 0 – 50 degrees C ambient
