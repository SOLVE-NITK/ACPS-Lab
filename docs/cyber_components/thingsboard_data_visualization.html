ThingsBoard Data Visualization

This section outlines the robust data visualization capabilities provided by ThingsBoard through its Web UI, dashboards, widgets, and the flexibility of custom widgets.

Web UI​

ThingsBoard's Web UI facilitates effective management and visualization of key business entities like Devices, Customers, Users, and Dashboards, each essential for structured IoT applications management.

Web UI Capabilities and Main Business Entities​

  • Devices: Devices connected to ThingsBoard, monitored and controlled by users.
  • Customers: Manage customer information and associate devices to specific customers.
  • Users: Define access levels within the ThingsBoard environment to maintain security and data privacy.
  • Dashboards: Tools for creating customized visual data representations from connected devices.
  • For more details, see the ThingsBoard Web UI guide.

Source: The Official ThingsBoard, Inc YouTube Channel

ThingsBoard User Roles and Permissions​

ThingsBoard supports various user roles with specific permissions, dictating access levels to features and data.

  • Discover more about user roles and permissions in the Tenant Profiles documentation.

Source: The Official ThingsBoard, Inc YouTube Channel

Dashboards​

Customizable interfaces in ThingsBoard allow users to visualize device data in real-time with various layouts and widgets configured to display data as charts, maps, gauges, etc.

  • Explore in-depth in the Dashboards documentation.

Source: The Official ThingsBoard, Inc YouTube Channel

Widgets​

Widgets are the building blocks of dashboards, offering dynamic data visualization:

  • Types: Charts, cards, maps, controls, etc.
  • Data Sources: Configurable to pull data from device telemetry, attributes, or APIs.
  • Settings: Includes basic and advanced configurations for customization.
  • Detailed widget information is available in the Widgets documentation.

Source: The Official ThingsBoard, Inc YouTube Channel

Custom Widgets​

For advanced customization, ThingsBoard allows the development of custom widgets to meet specific visualization needs.

  • Learn about developing custom widgets from the Custom Widgets Development guide.
