# ThingsBoard Platform Basics

## Platform Overview
### Features and Architecture
ThingsBoard is an open-source IoT platform that supports a multitude of devices and offers extensive customization capabilities. It is designed to handle millions of devices and their data, providing scalability and robustness. The architecture supports detailed management of device connectivity, data collection, and visualization.
- [Learn more about ThingsBoard architecture](https://thingsboard.io/docs/getting-started-guides/what-is-thingsboard/)

### Platform Capabilities and Functionality
The platform enables users to manage devices, collect and visualize data, and perform analytics with built-in or custom widgets. It supports multi-tenant installations, robust security configurations, and comprehensive APIs for deeper integration.

## Web UI
### Basic Web UI Capabilities and Main Entities
The ThingsBoard Web UI provides an intuitive interface to manage Devices, Customers, Users, and Dashboards, allowing for efficient IoT ecosystem management.
- [Details on entities and relations](https://thingsboard.io/docs/user-guide/entities-and-relations/)

### Available Widget Types and Configuration
The platform supports a range of widgets including charts, maps, gauges, and controls, which can be customized to suit various IoT applications.
- [Configure widgets and dashboards](https://thingsboard.io/docs/user-guide/dashboards/)

## ThingsBoard API

The ThingsBoard API facilitates device connectivity and integration with server-side applications, supporting a variety of network protocols and enabling extensive system interactivity.
- [Learn more about ThingsBoard API](https://thingsboard.io/docs/api/)

## Rule Engine

ThingsBoard's rule engine processes messages through a configurable set of rules, enabling dynamic decision-making and action triggering based on device data. The rule engine utilizes messages, relations, and nodes to manage device communications and actions, each with different implementations to meet specific requirements.
- [Getting started with the rule engine](https://thingsboard.io/docs/user-guide/rule-engine-2-0/re-getting-started/)
